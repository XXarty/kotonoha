# KOTONOHA 素雅动态背景设计

## 目标

在现有暖白和纸底色上增加几乎不打扰阅读的环境动态，让页面更有呼吸感，同时保持 KOTONOHA 的安静、温情与高级感。

## 已确认方案

采用“和纸微尘 + 淡墨圆影”：

- 全站背景保留现有 `mist → paper` 渐变。
- 使用纯 CSS 叠加低对比度点纹，不新增图片请求。
- 在页面左右边缘放置两层大尺寸、低透明度的墨绿/鼠尾草圆影；圆影不得覆盖为可识别图标或干扰内容。
- 两层圆影以不同的 30–45 秒周期做缓慢、小范围漂移，避免同步和机械感。
- 动态背景位于内容层之后，不拦截点击，不创建横向滚动，也不改变现有卡片、排版或导航结构。

## 动效与可访问性

- 动画仅使用 `transform` 与 `opacity`，避免持续触发布局计算。
- `prefers-reduced-motion: reduce` 下停止漂移，保留静态纹理。
- 正文与表单继续使用不透明或高可读背景；动态层透明度必须足够低，不改变现有文字对比度测试结果。
- 手机端缩小圆影并降低可见度，确保首屏文字清晰。

## 单词层级说明

“日常核心”和“进阶扩展”继续作为说明性信息区，不增加链接或筛选跳转。实际学习入口仍为下方按词性分类，避免制造尚未定义的层级导航行为。

## 验收标准

- 桌面与手机端可见柔和点纹和缓慢圆影，但文字始终是视觉主体。
- 页面无横向溢出，背景层不响应鼠标或触摸。
- 减少动态模式下背景动画关闭。
- 现有单元测试、类型检查、生产构建和关键 E2E 继续通过。
